Import Model from Database as LINQ entities

Add a BCS model from a database and generates the necessary LINQ classes to retrieve and change the data. The recipe will also generate a LINQ-to-SQL file (.dbml) from the database.

Arguments

Name Description
Model Settings
Name Enter the name of the BCS Model.
Display Name Enter the display name of the BCS Model. This name is visible in dialogs to the user.
Estimated Item Count Enter number of estimated count of elements in the data source.
Is Cached Check if the data should be cached.
Database Settings
Server Enter the name of the database server.
Database Enter the name of the database.
Permissions
Users Enter all users which should have permission to this model, separated by ';', e.g. 'NT AUTHORITY\Authenticated Users' or 'STS|SecurityTokenService|http://sharepoint.microsoft.com/claims/2009/08/isauthenticated|true|http://www.w3.org/2001/XMLSchema#string'.
