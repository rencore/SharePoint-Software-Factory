System Requirements

The SPSF SharePoint Software Factory supports following configurations:

Visual Studio 2012 Visual Studio 2010 Visual Studio 2008
SharePoint 2013 yes no no
SharePoint 2010 yes yes yes
SharePoint 2007 no yes yes

Prerequisites

Visual Studio 2012

  • Visual Studio 2012
  • SharePoint 2010/2013
  • OpenGAX
  • MS Office Developer Tools for Visual Studio 2012
  • SharePoint Software Factory 4.2

Visual Studio 2010

  • Visual Studio 2010
  • SharePoint 2007/2010
  • GAX 2010
  • SharePoint Software Factory 3.2

Visual Studio 2008

  • Visual Studio 2012
  • SharePoint 2007/2010
  • Microsoft Guidance Automation Extensions
  • Guidance Automation Extensions Update (GAXP1409)
  • SharePoint Software Factory 3.2
